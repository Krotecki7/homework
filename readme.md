# Cерверная часть виджета банковских операций.

Виджет имеет возможность отображать операции клиента.

## Зависимости:

flake8 = "^7.1.0"
black = "^24.4.2"
isort = "^5.13.2"
mypy = "^1.10.1" 

## Функции в программе:

1. Функции для максировки номера карты и лицевого счета
2. Функции для обработки данных пользователей
